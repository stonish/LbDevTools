### Added

- Scripts:
  - `lb-project-init`
  - `lb-dev`
  - `lb-devtools-datadir`
  - Git custom commands

    - `git lb-checkout`
    - `git lb-push`
    - `git lb-clone-pkg`
- Generic makefiles for CMT and CMake
- CMake support modules (from [Gaudi v30r2][])
